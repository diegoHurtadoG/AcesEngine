Detailed Description

Defines a fast way to make a card, ideal to board games.

The cards in this class can have different assets, in the assets directory there is a pre loaded card tileset, the setters and getters are public and it has the draw method, the turn over method is in process. The drag and drop is another method outside the class

Has a Drawable component
